# Руководство пользователя

> **ScienceJournal — Платформа публикации научных статей**  
> Версия 1.0 | Группа ПИЖ-б-о-24-1 | СКФУ, 2026

---

## Введение

Данное руководство предназначено для конечных пользователей платформы ScienceJournal. Документ описывает порядок работы для двух типов пользователей:
- **Читатель (Reader)** — просматривает и читает публикации
- **Публицист (Publisher)** — публикует научные статьи + все возможности Читателя

---

## Системные требования

| Компонент | Требование |
|---|---|
| Браузер | Google Chrome 90+ / Mozilla Firefox 88+ / Safari 14+ |
| Интернет-соединение | Требуется (подключение к серверу) |

---

## 1. Начало работы

### 1.1. Открытие платформы

Введите в адресной строке браузера адрес платформы:
```
http://localhost:5173
```
*(При использовании на учебном сервере адрес укажет преподаватель)*

Вы увидите **приветственную страницу (Welcome)** с кнопками «Войти» и «Зарегистрироваться».

---

### 1.2. Регистрация нового пользователя

1. На приветственной странице нажмите **«Зарегистрироваться»**
2. Заполните форму регистрации:

| Поле | Описание | Пример |
|---|---|---|
| Полное имя | Ваше имя и фамилия | Иван Петров |
| Логин | Уникальный логин (до 12 символов) | ivan2026 |
| Пароль | Ваш пароль (до 45 символов) | •••••••• |
| Пол | Выберите из списка | Мужской |
| Роль | **Читатель** или **Публицист** | Публицист |

3. Нажмите кнопку **«Зарегистрироваться»**
4. При успехе — вы автоматически войдёте в систему и попадёте в ленту публикаций

> **Важно:** логин должен быть уникальным. Если введённый логин уже занят, система покажет уведомление об ошибке.

---

### 1.3. Вход в систему

1. На приветственной странице нажмите **«Войти»**
2. Введите ваш логин и пароль
3. Нажмите **«Войти»**
4. При успехе — вы попадёте в ленту публикаций

---

## 2. Просмотр публикаций (для всех авторизованных)

### 2.1. Лента статей (Explore)

После входа вы оказываетесь на странице **Explore** — ленте научных публикаций.

На странице отображается список публикаций. Каждая карточка содержит:
- Обложку статьи (если загружена)
- Заголовок статьи
- Краткое описание
- Имя автора
- Дату публикации

**Пагинация:** для перехода между страницами используйте кнопки навигации внизу списка.

---

### 2.2. Чтение полной статьи

1. В ленте нажмите на карточку интересующей публикации
2. Откроется страница **Read** с полным содержанием:
   - Обложка
   - Заголовок
   - Автор и дата публикации
   - Полный текст описания
   - Кнопка **«Скачать PDF»** (если PDF загружен автором)
3. Нажмите **«Скачать PDF»** для скачивания файла статьи

---

### 2.3. Возврат к ленте

Используйте **Header** (верхнюю навигационную панель) для перемещения между разделами:
- **Главная** — возврат на ленту
- **Выйти** — завершение сессии

---

## 3. Публикация статей (только для Публицистов)

> Раздел доступен только пользователям с ролью **Публицист**.

### 3.1. Создание новой публикации

1. В Header нажмите **«Опубликовать»** или перейдите на страницу `/publish`
2. Заполните форму:

| Поле | Описание | Ограничение |
|---|---|---|
| Заголовок | Название вашей статьи | до 45 символов |
| Описание | Текст статьи или аннотация | до 700 символов |
| Обложка | Изображение для карточки (jpg, png) | Необязательное |
| PDF-файл | Полный текст статьи в формате PDF | Необязательное |

3. Нажмите **«Опубликовать»**
4. При успехе — появится уведомление об успешной публикации
5. Новая статья появится в ленте Explore

---

### 3.2. Ошибки при публикации

| Сообщение | Причина | Решение |
|---|---|---|
| «Заполните все обязательные поля» | Не введён заголовок или описание | Заполните все поля формы |
| «Нет доступа» | Ваша роль — Читатель | Зарегистрируйтесь как Публицист |
| «Ошибка авторизации» | Сессия истекла | Войдите в систему заново |

---

## 4. Завершение работы

### 4.1. Выход из системы

1. В Header нажмите **«Выйти»**
2. Ваша сессия будет завершена
3. Вы будете перенаправлены на страницу входа

> **Важно:** после выхода ваши данные авторизации удаляются из браузера. Для продолжения работы потребуется повторный вход.

---

## 5. Часто задаваемые вопросы

**Q: Могу ли я изменить роль после регистрации?**  
A: Нет, роль выбирается при регистрации и не меняется через интерфейс.

**Q: Я не вижу кнопку «Опубликовать» в меню.**  
A: Эта функция доступна только Публицистам. При регистрации убедитесь, что выбрали роль «Публицист».

**Q: PDF не открывается после нажатия «Скачать».**  
A: Убедитесь, что автор загрузил PDF при публикации. Если поле PDF пустое — файл не был загружен.

**Q: Сессия завершилась сама по себе.**  
A: JWT-токен имеет ограниченный срок действия. Войдите в систему повторно.
