## Оценка трудозатрат (COCOMO)

### Объём кода

| Компонент | Строк кода (LOC) |
|---|---|
| Backend (JS) | ~600 LOC |
| Frontend React (JSX/JS) | ~1200 LOC |
| CSS стили | ~500 LOC |
| **Итого** | **~2300 LOC** |

### Расчёт по COCOMO Basic (Organic mode)

```
E  = a × (KLOC)^b = 2.4 × (2.3)^1.05 ≈ 5.9 person-months
D  = c × E^d     = 2.5 × (5.9)^0.38  ≈ 5.2 months
P  = E / D       ≈ 1.1 persons

Для учебного проекта (1 разработчик, часть времени):
Фактическое время = 10 недель ≈ 2.5 месяца
```

## Управление рисками

| # | Риск | Вероятность | Влияние | Митигация |
|---|---|---|---|---|
| R1 | MongoDB недоступна (нет подключения) | Средняя | Высокое | `.env`-конфигурация, graceful shutdown |
| R2 | JWT истёк, нет refresh | Высокая | Среднее | Перенаправление на /auth при 401 |
| R3 | Загруженный файл — не PDF/image | Средняя | Среднее | Валидация MIME-типа через Multer |
| R4 | CORS-ошибка при развёртывании | Средняя | Высокое | Вынести origin в `.env` |
| R5 | MongoDB Duplicate Key при регистрации | Высокая | Низкое | Catch DuplicateKey → HTTP 409 |
